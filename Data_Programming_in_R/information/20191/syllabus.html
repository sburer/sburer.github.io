Course Description and Goals. This course introduces the principles and practices of computer programming that are applicable to software development in any programming language, but the course places a special emphasis on techniques and tools useful for data scientists. Accordingly, the main programming environment for the course will be the open-source R environment using RStudio. The main topics include:

  • Introduction to R and RStudio
  • Basics of Data Frames
  • Debugging
  • Data Transformation and Cleaning
  • Summary Tables
  • Graphics
  • External Data Sources
  • Dynamic Reports and Web Apps
  • Intermediate Data Types
  • Control Logic
  • Functions
  • Error Handling

Course activities will include formal and active-learning lectures, individual and group in-class exercises, individual homeworks, individual tests, and a group project. In addition, students are expected to prepare ahead of time for each class meeting.
