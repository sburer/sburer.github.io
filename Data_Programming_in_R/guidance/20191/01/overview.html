This module, we begin our journey to become better computer programmers and to gain skills in R, and to do so, we familiar ourselves with of the the tools we’ll be using throughout the course: R, RStudio, and GitHub. Our specific learning objectives are to:

  • Become familiar with RStudio’s user interface.
  • Use R as a simple calculator.
  • Create simple plots in R.
  • Manage packages in R.
  • View and set the working directory in R.
  • Demonstrate working knowledge of R scripts within RStudio, including:
    • Creating and sourcing an R script without error on your own computer.
    • Preparing an R script to be shared with another person so that it can be sourced on their computer without error.
    • Sharing an R script through GitHub.

These learning objectives will be assessed via Quiz 1 and Homework 1.
