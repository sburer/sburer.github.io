Classifying Data

  • Imagine data in a single column/variable

  • Two complementary classifications

    • Measurement scale (ratio vs categorical)
    • Time collection (cross-sectional vs time-series)

Measurement Scale

Ratio data

  • The most common type of data
  • Numerical
    • Ratios, “natural zero”
    • Differences
    • Order
  • E.g., sales figures, task completion times
  • Highly amenable to statistical analysis

Measurement Scale (cont’d)

Categorical data

  • Occurs frequently
  • Sorts data into groups (usually without order)
  • Does not make sense to talk about
    • Ratios
    • Differences
  • E.g., zip code, race
  • Statistical analysis is more limited

Time Collection

Cross-sectional data

  • Data that is collected at one moment in time
  • For example
    • proportion of current delinquent accounts
    • tonight’s warehouse inventory levels
    • customer demographic information

Time Collection (cont’d)

Time-series data

  • Data of the same type that is collected over time (usually at equally spaced intervals)
  • For example
    • price of strawberries over the last 12 months
    • semester-by-semester MBA enrollment
  • Cross-sectional data becomes time series data if it is collected more than once!
