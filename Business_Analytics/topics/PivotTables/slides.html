PivotTables

  • A PivotTable is a tool to summarize a large amount of data quickly

  • Any PivotTable can easily be turned into a PivotChart

  • LinkedIn Learning tutorial for constructing PivotTables

Example: Use PivotTables to answer the following about data in the Excel file homes.xlsx

  • What is the average price for homes in below normal, normal, and above normal conditions?

  • Which bed/bath combination is most common?

  • Count the number of homes in the following price groups: $0-$20000, $20000-$40000, $40000-$60000, etc. Also prepare a chart
